# Anandmaya - React Spiritual Website

A beautiful, responsive React-based website for "Anandmaya" - a spiritual sanctuary dedicated to devotion, meditation, and community.

## 🌟 Features

- **React 18** - Modern React with hooks and functional components
- **React Router** - Client-side routing for seamless navigation
- **Framer Motion** - Smooth animations and transitions
- **Responsive Design** - Works perfectly on all devices
- **Interactive Components** - Dynamic forms, hover effects, and user interactions
- **Modern UI/UX** - Clean, spiritual-themed design with smooth animations

## 🎨 Design Elements

- **Color Scheme**: Warm earth tones with gold accents (#8B4513, #D2691E, #FFD700)
- **Typography**: Playfair Display for headings, Inter for body text
- **Icons**: Font Awesome icons for visual elements
- **Animations**: Smooth transitions and scroll-triggered animations

## 📱 Pages & Components

### **Pages:**
1. **Home** - Landing page with hero section and service overview
2. **Online Pooja** - Dedicated page for pooja booking and information

### **Components:**
- **Navbar** - Responsive navigation with mobile menu
- **Hero** - Animated hero section with service options
- **ServiceSection** - Reusable service display component
- **Contact** - Contact form with validation
- **Footer** - Site footer with links and social media
- **PoojaHero** - Online Pooja page hero
- **PoojaTypes** - Interactive pooja type cards
- **HowItWorks** - Step-by-step process explanation
- **LivePooja** - Live streaming information
- **BookingForm** - Complete booking system with validation
- **Testimonials** - Customer reviews and testimonials

## 🚀 Getting Started

### Prerequisites
- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start development server:**
   ```bash
   npm start
   ```

3. **Open your browser:**
   Navigate to `http://localhost:3000`

### Build for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.
